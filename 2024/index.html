# 24 年年度总结

今天是 2025 年 4 月份下旬，早就想说写一篇来总结一下 2024 年了…… 结果一直忘记…… 现在终于可以来写这个了

自从 2023 年我自杀出院后我就放平心态开始摆烂了。但是由于我休学，在家的话的确没有什么事可以做，因此…… 我就只能给自己找事干了

或许…… 这也不错？

# 没有抑郁的的感觉真好

由于药物的控制，我每天都基本上很开心。没有抑郁发作的我情绪变得同正常人无异。但是药物还是会让我的脑子变得晕乎乎的，即使有安非他酮，我也没有办法

每天不能午休，让我的作息打乱了…… 还是多少有点难受…… 不过好在我会逐渐适应

那学校不能去，在家又有啥意思呢？我在我家这里是没有什么地方可以去，也没有朋友。我想了想…… 我可以准备考试

高数 英语 和 C 这三门。一想到自己还有 2 年多的时间，就开始慢慢学了…… 但是我的底子是只有小学生学历啊… 这确实有点难为我。不过我也说了，摆烂嘛…… 能学多少是多少，不要给自己太多压力。父母自从我自杀后也说：你身心健康，每天开开心心的活着就可以了。不要求你别的了。算是很开放的了

# 自学 - 相信自己

当时想想哈…… 一开始的安排是我上午学 C，下午休息什么的…… 但是很可惜，刚开年我就发烧了，住院输液…… 这不是一个好头啊（

# 第一季度 - 冬到春

# Jan.

一月份整体还是玩的比较多… 沉迷于田野打架 5 (战地 V)

这段时间的减肥也是没有闲着，我开始少吃，少吃，少吃。然后慢慢减肥

一月初没记错的话应该有 180 斤这样？

### Feb.

二月份，我学到了函数递归。说实话我现在也不太会递归，它的参数传参太复杂了有点让我难以理解。不过还好，递归嘛…… 递多了容易压栈…… 用处也不是特别特别大就是

那会已经从巨胖 -> 比较胖了，还是胖。穿裙子什么的还是……e…… 算了 不想回忆了

2.19 我的情人节礼物终于到了。他给咱买的一盒巧克力 (其实吧…… 当时在减肥…… 我说了不要让他买，结果最后还是买了) 粉色的德芙盒子，真的超级好看！

由于我这个月一直犯病，就很早之前的心脏右下方压迫痛。所以又去了省医院做了一堆检查… 挂了疼痛科，心内科也没查出什么毛病来。一切都正常

也算是自杀后做了个全面体检了

我那会怎么天天玩战地 V 啊 简直草了

### Mar.

三月份，我学到了指针。在学 C 之前我就听说了，指针很难。然后我学完了也没觉得这玩意有多难啊…… 搞不太懂了……

这段时间也开始背英语单词，学习英语。不过速度嘛…… 不能说是老牛拉破车吧…… 那也是慢到龟速了

3 月初有亲戚的女儿嫁人了…… 那家是很富的那种，所以举办的也是特别豪华。第一次吃到那么多海鲜，什么佛跳墙，鲍鱼什么的…… 甚至我都不会吃……

然后我开始学数学了。由于我学到初一下册的一元一次不等式就没在去学校了。因此我只能从初中学起，直到学到大学的高等数学

我相信我自己…… 我相信自己可以的……

现在就是上午学数学，下午…… 下午…… 我怎么还在玩游戏啊喂！

3 月中旬。他送了我一套键帽。非常好看的冬天主题。粉蓝白配色 什么刻板印象 总之还是很喜欢哒！

由于他去年体育课选的是跳绳，他说这玩意真的巨燃脂，让我也来跳绳减肥。于是……(后来来觉得这真是一个错误的决定……)

# 第二季度 - 春到夏

# Apr.

4 月份…… 依然保持上午学数学，下午学 C…… 不过好像学一会我就去打游戏了…… 算了……

我是这样的… 真的有够摆

妈耶…… 为什么我还是一直在玩游戏啊喂！

减肥呢还是在正常进行。不过 4 月份天气暖和了就早上起来去跑步，然后中午吃饭前跳绳 20 分钟。这下出问题了

由于我吃药，安非他酮导致我无法睡午觉。而晚上的齐拉西酮又会导致嗜睡。所以这意味着我晚上必须早睡。早上真的很困起不来，很痛苦。只能要么跑完回来睡觉，要么一上午昏昏沉沉的。我当时没有发现安非他酮的提神作用，如果早知道就可以早上一醒来就吃安非他酮了……

你可能说：啊为什么不晚点去跑？

很简单：我这里…… 公园是没有的，路也不平，都是很陡的坡。唯一有的一条相对不算很陡的路，但是那条路属于五不管地带，路没人修，然后都是灰尘，都是土。早上车少，所以没那么灰尘。如果有车从你旁边经过，那大概率会弄你一身子土

这真的太难受了…… 我真的真的真的！非常讨厌这个破地方

# May.

5 月份，我的生日 + 520 都在本月了。于是我让他提前给我买了礼物。就是我一直喜欢的那条裙子 就是经典药娘裙 ()……

然后到手真的是非常期待，因为这是我梦寐以求的裙子…… 真的是特别喜欢…… 问了客服，腰围 74 应该选择 M 码还是 L 码，因为 M 码是 68-74，L 码是 74-80。我怕买回来太大 / 太小，于是我询问了客服

客服推荐我买 M，于是买回来发现…… 果然扣子是扣不上的

但是拉链也可以拉上，这样就还算勉强……？然后看着照片里的自己，这是自己？由于那会我的腿还是蛮粗的 (那会大约 140 斤吧) 于是我使用了瘦腿，P 了一下。然后看看效果…… 简直绝了…… 这真的是我吗？我之前很胖的时候根本没办法 P 腿的…… 现在可以 P 了…… 而且…… 真的很细

我当时就想，这就是我的目标，我要以这个为目标继续加油努力往下减！

这段时间内我已经将 C 学完了，开始尝试写代码来巩固自己的学习成果。感觉自己好像…… 也并不笨。还是蛮聪明的嘛

数学已经学到初三了，开始接触如反比例函数一类的内容。没想到自己一个小学生还能学懂函数……

# Jun.

6 月份，天气也开始热了，所以停止掉运动减肥，改为普通的节食减肥。也就是说，减肥就到此结束。从 2023 年 3 月 8 日开始到 2024 年 6 月，共约 1 年 (其中有 2 个月没有减肥) 的时间从 112.5KG 减到 120 多斤。没有想到自己居然能够成功做到

这段时间内还是依旧保持上午数学下午实操 C，我使用 C 写了一个计量我还有多少药物的程序，虽然很简陋，不过可以计算自己的精神病药物什么时候吃完，还有多少颗药

六月份，我学到了高一的数学，开始学习集合等等。感觉自己的进度还是蛮快的。毕竟自己只需要学高数而不需要学几何，学起来还是很快的

# 第三季度 - 热热热

# Jul.

7 月份，我的节食减肥还在继续，学习也没有拉下

由于 C 我自认为学的差不多了，因此在本月中旬就开始学习 C++

C++ 感觉也不是特别特别难就是了……

在此之前我发现我在写 C 上投入了蛮多时间的…… 拿来自己瞎写。现在看到我那个拿来测试代码的项目已经写了一万行 C 代码了……

# 换宽带

由于移不动实在是恶心至极，于是我就决定去换一家运营商。一开始决定是换电信的，后来问了一大推营业厅的工作人员连公网 IPV6 地址是啥都不知道…… 绝了

最后听说我可能要投诉他们，就不让我办了

我就跑去隔壁的联通……109 一个月，千兆。办下来了

然后我才发现…… 我家这个是公网 V4！不仅公网 V6，甚至有公网 V4

这可太惊喜了……

于是我马上整了一个域名，并且购买了一些服务器 “洋垃圾”。准备折腾折腾

在这段期间感谢我的好朋友 —— 杉杉，是他带我教我如何折腾

那会我在计网方面还是个废物… 不过既然要整服务器了，那肯定不能用 Windows 了，要用就用 Linux！

听他的选择使用了 Ubuntu

然后也是折腾了一段时间，不过接下来……C++ 得暂停了。得先去学一下 Linux 和 Docker 了……

# Aug.

八月份，正值酷暑…… 于是我赶快开始学习 Linux 指令和 Docker 指令相关的内容，先把 C++ 搁置了

好歹之前有使用过一点点 Linux，因此学起来还是很快的

然后就是我发现我的服务器根本没有办法 24H 运行，我服务器的 CPU 使用的是 2666V3，导致整机待机都有 20-30W 的功耗。再怎么说我还在家里呆着，我也没有多少收入。不想听我妈的唠叨 (讲电费的事情)

月中，我就把 Docker 也学完了，于是继续回去学习 C++

8 月底… 我和他 (我对象) 吵架了。他整整 5 天没有理我…… 这种事情很常见，上次冷战是 4 月份的时候。那会就对他这种行为开始免疫了。所以这次我也干脆不理他，他爱咋地咋地

# 开始 HRT

当然…… 虽然说是免疫…… 可是心里还是会难受。再怎么说也是谈了快 2 年了… 而正好那段时间我脸上不知道为啥一直长痘痘，加上自己其实…… 也想要更像女孩子一点。于是在群里认识倒了我姐…

他带我如何买色普龙，我买了一盒国色。并且一到手就开始吃了 12.5mg。之后每隔 2 天吃 12.5。相信这能够给我带来足够的变化吧！

月底，我学到了三角函数。其实感觉也没有那么复杂，就是题目有点绕。理解了的话还是蛮容易的，至少我自学能学进去……

最后，我使用 C 重构了我的药物倒计时程序。这下…… 我可以说是把 C 也看完了 会不会再说啦～

# Sep.

本月…… 体重逐渐来到 120 斤及以下。我的减肥还是很成功的。看到镜子中的自己，穿着以前特别肥大的衣服…… 感觉很可笑。只有我知道我自己成功了，并且从 225 斤减到 120 斤的成就感也是我这辈子绝无仅有的……

本月，我开始实现我的 C++ 项目：计算器

我想做一个计算器，类似于手机上的计算器，并且在这个计算器上投入了很多时间

而这段时间我甚至还在学三角函数……

# 第四季度 - 最爱的秋冬

# Oct.

十月份，本月打算去北京医院看看我的病。定在了 10 月 14 号

而在这之前…… 我开始学 HTML CSS JS

在到北京前，已经学到了 JS 函数……

到了北京，这也是我第一次来北京，并且是一个人

14 号 到了北京累了，出去找了一下医院就直接休息了

15 号 上午看了一下大夫，做了一些检查。具体可以看 抑郁 - 自杀 - 死亡 这篇文章；很快就看完大夫然后去逛了逛南锣鼓巷，感觉就和我前任说的 (我前任是北京的正儿八经的京爷) 一样，确实没啥意思。下午去北海公园。在回酒店的路上看到了车牌号是 VA 开头的车，回去一查才发现是官老爷的车（

16 号 今天预约了故宫。大早上就去排队，不过我一个人，加上对历史…… 属实不懂啊… 所以也感觉没啥意思就想说既然预约了故宫可以去天安门广场，那就去看看咯

于是走着去天安门广场，这个路看着很近其实很远。主要是还担心自己走错路…… 中午的时候终于到了。先是看到了天安门城楼，嗯…… 怎么说呢？和电视里看的一样（

然后走到长安街西侧，有个底下通道可以到达天安门广场上。站在广场上远远看到了人民大会堂，想说，下次一定去里面看看！旁边是国家博物院，以及…… 南面一堆人排队的老毛纪念堂……

回到酒店后，感觉自己有些累。加上早上吃了两颗安非他酮，其实压根睡不着…… 所以就躺了一会，没有出去玩

也就是这天决定…… 我要开始吃补了，感觉在这样下去一直双低也不是个事啊……

17 号 要准备回家了。但是昨晚由于安非他酮我压根没有睡好…… 于是…… 安非他酮…… 哎……

19 号 吃下第一颗补佳乐 真的是甜甜的…… 真的和糖果一样……

回到家就继续学习 JS 了，还是和之前一样 做自己喜欢的事情

然后发现自己胖了 3 斤多……

# Nov.

这可能是我最平淡的一个月了……

前端什么的写完了，正好遇到我和他 2 周年纪念日，于是就用 HTML+CSS+JS 写了一个很丑的页面…… 嘛…… 至少是自己亲自做的嘛！

月底，我打算重写一些之前写的计算器，重新拉一坨大的

然后…… 还有就是？我把我的手机 K50 刷了 HyperOS 2。不得不说 Hy2 还是蛮丝滑的，相对我之前使用的 MIUI13 来讲

减肥…… 我摆烂了…… 欸嘿！

# Dec.

2024 的最后一个月，月初，我去查了一下激素…… 想说我吃糖这么久了，雌二醇应该上来了吧？(反正体重是上来了不少哈)

于是我 12.1 当天吃了 4mg 补佳乐，下午吃了 12.5mg 色普龙…… 以下是结果

  • 雌二醇 13.83pg/ml
  • 睾酮 <0.069ng/ml
  • 泌乳素 12.17ng/ml

嗯…… 正常来说…… 身为 mtf 应该把激素控制在：

  • 雌二醇 50-150pg/ml
  • 睾酮 <0.55ng/ml
  • 泌乳素 4.79-23.3ng/ml

真是绝了！

所以我只能联系另一位姐姐 (其实年龄比我小) 然后换凝胶……

12.8 日，由于我很隐晦的说了一句上面，所以我的 QQ 号被封了 30 天。于是换使用 TG

这个月的减肥基本上就是摆烂的…… 体重…… 算了……

这个月我继续开发我的计算器 2.0，以及去学习了 Git 的用法，并且开始研究如何使用 Github

月底，应该是我本年最忙的几天了

下午：开发 UHC_BE，是一个基岩版的脚本系统

然后开发累了的话就去开发我的计时页面 (使用前端)。然后我发现我开发这个页面只花了 6 个小时…… 嗯…… 这个满足感还是很强的

还要准备在明年 1 月份等他放假后去找他见面，两年一直以来都是网恋… 终于要见面了

这个月的月底简直… 最忙了！
